网络和后台接入

1.网站请求察看
F12 找到network 刷新找到 html 选中后 点response 可以查看后端请求的信息 url和地址栏一样 察看请求方式 有 get post
（网页里面 link img 等请求都会出现在network 页面）
信息详情里 有和url一一对应的数据名称 这就是请求查询的姿势
2.后台请求实现
请求数据 常用xhr （json） 通过js发起请求获得数据
以用axios 为例
请求姿势 axios.get(“url”).then(function(response){
console.log(response)
})
对数据进行请求和引用

首先引入axios
import axios from “axios”

然后data（） 同级

this.getit()
},

}
